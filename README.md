# V2raya для роутеров Xiaomi 🚀

Репозиторий содержит инструкции и скрипты для развертывания системы пакетов V2raya на роутерах Xiaomi. Это позволяет значительно расширить функционал устройства, устанавливая полноценное ПО.

-------------------------------------------------------------------------------
🛠 ПОДГОТОВКА
-------------------------------------------------------------------------------
Перед началом убедитесь, что выполнены следующие условия:
1. SSH доступ: На роутере должен быть открыт доступ по SSH.
2. Накопитель: Подключена USB-флешка или внешний диск (рекомендуется файловая система ext4).
3. Интернет: Роутер имеет стабильный доступ в сеть.

-------------------------------------------------------------------------------
📦 УСТАНОВКА
-------------------------------------------------------------------------------
Для запуска процесса установки выполните команду в консоли роутера:
```bash
curl -L -k -s https://raw.githubusercontent.com/frogost/v2raya_xiaomi/main/setup_v2raya.sh -o /tmp/setup_v2raya.sh && chmod +x /tmp/setup_v2raya.sh && /tmp/setup_v2raya.sh install
```
-------------------------------------------------------------------------------
📋 УПРАВЛЕНИЕ И ЛОГИ
-------------------------------------------------------------------------------
После установки вам будут доступны следующие возможности:

* Установка: ```/data/v2raya/startup_v2raya.sh install``` (если вы скачали скрипт setup_v2raya.sh вручную)
* Обновление v2raya и ядра xray: ```/data/v2raya/startup_v2raya.sh update```
* Удаление: ```/data/v2raya/startup_v2raya.sh uninstall```
* Лог установки: ```/tmp/v2raya_install.log``` (доступен до перезагрузки)
* Лог автозапуска: Внутри папки v2raya (например, ```/mnt/usb-XXXX/v2raya/v2raya_install_final.log```)

-------------------------------------------------------------------------------
⚙️ НАСТРОЙКА В ВЕБ-ИНТЕРФЕЙСЕ V2RAYA
-------------------------------------------------------------------------------
После открытия интерфейса при первой установке вам нужно будет задать постоянный логин и пароль. Далее в правом верхнем углу будут доступны следующие разделы:
* Настройки — редактирование вашей маршрутизации.
* Журнал — логирование в реальном времени.
* Язык — изменение локализации.
* Пользователь — текущий пользователь, под которым вы вошли.

Пример настройки в интерфейсе:
Вы можете воспользоваться этими параметрами для настройки маршрутизации:
* Прозрачный прокси / Системный прокси: Включить режим разделения трафика
* Реализация прозрачного прокси / Системного прокси: tproxy (для маршрутизирования tcp и udp соединений)
* Excluded Interface Prefixes: eth*
* Режим разделения трафика на порте с правилами: RoutingA
* Предотвратить DNS-спуфинг: Расширенная настройка
* Специальный режим: Выкл
* TCPFastOpen: По-умолчанию
* Уровень журнала: Info
* Сниффер: Http+TLS+Quic
* Мультиплекс: Выкл
* Автоматически обновлять подписки: Обновлять подписки регулярно (в часах)
* Цифра обновления (в часах): 6
* Режим обновления подписок и GFWList: Следовать за Прозрачным прокси/Системным Прокси

Базовая настройка RoutingA:
Скопируйте этот текст и вставьте в настройки RoutingA:
```bash
# BLOCK TRAFFIC
#network(udp) && port(443)->block
domain(geosite:category-ads)->block

# SOURCE TRAFFIC
#source(192.168.31.15/32)->proxy

# DOMAIN GEO TRAFFIC
domain(geosite:yandex)->direct
domain(geosite:youtube)->proxy
domain(geosite:google-gemini)->proxy
domain(geosite:telegram)->proxy
domain(geosite:whatsapp)->proxy
domain(geosite:discord)->proxy
domain(geosite:ru-blocked)->proxy

# IP GEO TRAFFIC
ip(geoip:yandex)->direct
ip(geoip:private)->direct
ip(geoip:telegram)->proxy
ip(geoip:ru-blocked)->proxy

# OTHERS TRAFFIC
#network(udp)&&port(50000-65535)->proxy

# DEFAULT TRAFFIC
default: direct
```
Полезные ссылки и проверка:

-------------------------------------------------------------------------------
⚠️ ВАЖНЫЕ ПРИМЕЧАНИЯ
-------------------------------------------------------------------------------
* Системная папка: Файлы для работы V2raya находятся в ```/mnt/usb-XXXX/v2raya/```.
* Автозапуск: Скрипт автоматически настраивает запуск всех сервисов при загрузке системы и копирует себя в ```/data/v2raya/startup_v2raya.sh```.
